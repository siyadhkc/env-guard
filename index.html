Why env-guard
Without env-guard
code with secret
↓ commit
↓ push to GitHub
↓ GitHub scans
↓ alert sent
↓ key already exposed ✖
With env-guard
code with secret
↓ commit attempt
↓ blocked on your machine ✔
fix locally
↓ push clean code
↓ nothing leaked ✔

GitHub Secret Scanning is your last line of defense. env-guard is your first.
It runs locally, offline, before the secret ever leaves your machine.
